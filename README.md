# Mini-Python-Compiler
1. Símbolos
Operadores Aritméticos: +, -, *, /, //, %, **
Operadores Relacionais: ==, !=, >, <, >=, <=
Operadores Booleanos (Símbolos):  &, |, ~, ^
Símbolos de Atribuição: =, +=, -=, *=, /=
Símbolos de tipos de dados [] para listas, {} para dicionários
Símbolo Identificador de Início de Bloco: : (dois-pontos)
2. Palavras-chave
Identificadoras de Blocos: if, else, elif, for, while, def, class, try, except
Tipos de Dados: int, float, str, bool, list, dict, tuple, set
Funções Built-in: print(), range(), len(), input(), type(), int(), float(), str()
Operadores Booleanos (Palavras): and, or, not
Demais Palavras: import, from, in, as, return, None, True, False, pass, break, continue
